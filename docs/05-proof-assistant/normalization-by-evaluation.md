# Normalisation par évaluation (NbE) dans Astra Core

Astra Core utilise la NbE pour :

- normaliser les termes,
- vérifier l’égalité,
- optimiser les preuves.

## Principe

1. interpréter un terme dans un modèle,
2. réifier le résultat en forme normale.

## Avantages

- performance,
- simplicité,
- cohérence.

## NbE dépendante

La NbE gère :

- les indices,
- les contraintes,
- les preuves.

## Utilisation

- réécriture,
- simplification,
- extraction.
