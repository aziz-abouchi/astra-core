# Pruning de contraintes dans Heaven Core

Le pruning de contraintes permet :

- d’accélérer la résolution,
- de réduire l’espace de recherche,
- d’améliorer l’automatisation.

## Techniques

### 1. Détection de contradictions
Élimination immédiate.

### 2. Simplification
Propagation logique.

### 3. Pruning dépendant
Indices normalisés.

### 4. Pruning distribué
Branches synchronisées.

## Visualisation

- graphes de contraintes,
- logs,
- métriques.
