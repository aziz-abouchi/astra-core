# Indexation de buts dans Heaven Core

L’indexation de buts permet :

- une recherche rapide,
- une réécriture efficace,
- une normalisation optimisée.

## Structures

- discrimination trees,
- fingerprint tries,
- indexation dépendante.

## Utilisations

- réécriture,
- unification,
- propagation de contraintes.

## Visualisation

- arbres indexés,
- logs,
- métriques.
