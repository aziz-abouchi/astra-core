# Réécriture modulo théories dans Heaven Core

La réécriture peut être effectuée modulo :

- associativité,
- commutativité,
- unités,
- arithmétique,
- équivalences logiques.

## Théories supportées

### 1. AC (Associatif–Commutatif)
Pour les opérateurs comme `+` ou `*`.

### 2. Théorie des égalités
Réécriture orientée.

### 3. Théorie arithmétique
Basée sur Presburger.

### 4. Théories dépendantes
Indices normalisés.

## Utilisations

- simplification,
- normalisation,
- preuves automatisées.

## Visualisation

- arbres de réécriture,
- logs,
- contextes.
