# Indexation des termes dans Heaven Core

L’indexation des termes permet :

- une réécriture rapide,
- une recherche efficace,
- une normalisation optimisée.

## Structures d’indexation

### 1. Discrimination trees

Pour les termes structurés.

### 2. Fingerprint tries

Pour les termes normalisés.

### 3. Indexation dépendante

Pour les termes avec indices.

## Utilisations

- réécriture,
- unification,
- recherche de lemmes.

## Avantages

- performance,
- frugalité,
- scalabilité.
