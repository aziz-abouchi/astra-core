# Stratégies de normalisation dans Heaven Core

La normalisation est essentielle pour :

- l’égalité définitionnelle,
- la réécriture,
- la simplification.

## Stratégies

### 1. Normalisation forte
Réduction complète.

### 2. Normalisation faible
Réduction en tête.

### 3. NbE
Normalisation par évaluation.

### 4. Normalisation dépendante
Indices normalisés.

## Visualisation

- arbres de réduction,
- logs,
- métriques.
