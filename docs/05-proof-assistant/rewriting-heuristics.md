# Heuristiques de réécriture dans Heaven Core

Les heuristiques améliorent :

- la performance,
- la précision,
- la pertinence des preuves.

## Heuristiques principales

### 1. Réécriture orientée
Choix de la direction optimale.

### 2. Réécriture contextuelle
Réécriture dans les sous-termes pertinents.

### 3. Réécriture dépendante
Indices normalisés avant réécriture.

### 4. Réécriture AC
Associativité–commutativité.

### 5. Réécriture modulo contraintes
Propagation logique.

## Visualisation

- arbres de réécriture,
