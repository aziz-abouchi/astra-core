# Induction automatisée dans Astra Core

L’induction automatisée permet :

- de prouver des propriétés inductives,
- de simplifier les preuves,
- d’accélérer la vérification.

## Techniques

### 1. Induction structurée
Automatique pour les types inductifs.

### 2. Induction dépendante
Indices normalisés.

### 3. Induction mutuelle
Pour les types mutuellement inductifs.

### 4. Induction–récursion
Support complet.

## Visualisation

- arbres d’induction,
- obligations générées,
- logs.
