# Élagage de buts (Goal Pruning) dans Heaven Core

L’élagage de buts permet :

- d’accélérer les preuves,
- de réduire l’espace de recherche,
- d’éviter les branches inutiles.

## Techniques

### 1. Détection de contradictions
Élimination immédiate des branches impossibles.

### 2. Réécriture précoce
Simplification avant exploration.

### 3. Filtrage par contraintes
Suppression des buts incompatibles.

### 4. Pruning dépendant
Élagage basé sur les indices.

## Visualisation

- arbres de buts,
- branches coupées,
- logs.
