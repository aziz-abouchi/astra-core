# Synthèse automatique de lemmes dans Heaven Core

La synthèse de lemmes permet :

- d’accélérer les preuves,
- de réduire les obligations,
- d’améliorer l’automatisation.

## Techniques

### 1. Analyse structurelle
Extraction de motifs.

### 2. Analyse logique
Propagation de contraintes.

### 3. Analyse dépendante
Indices normalisés.

### 4. Synthèse distribuée
Recherche parallèle.

## Visualisation

- lemmes générés,
- arbres de preuve,
- logs.
