# Scheduling de buts dans Astra Core

Le scheduling de buts permet :

- d’optimiser la recherche,
- de réduire les divergences,
- d’améliorer l’automatisation.

## Critères de scheduling

- complexité,
- dépendances,
- contraintes,
- heuristiques.

## Stratégies

### 1. BFS logique
Exploration en largeur.

### 2. DFS contrôlé
Exploration en profondeur avec budgets.

### 3. Scheduling dépendant
Indices normalisés.

### 4. Scheduling distribué
Branches réparties entre nœuds.

## Visualisation

- arbres de buts,
- priorités,
- logs.
