# Évaluation partielle dans Astra Core

L’évaluation partielle permet :

- d’optimiser,
- de simplifier,
- de spécialiser.

## Principe

1. évaluer les parties connues,
2. conserver les parties inconnues,
3. générer un terme optimisé.

## Utilisations

- optimisation,
- compilation,
- preuves.

## Évaluation partielle dépendante

Les indices peuvent être :

- normalisés,
- simplifiés,
- résolus.

## Avantages

- performance,
- frugalité,
- réduction du bytecode.
