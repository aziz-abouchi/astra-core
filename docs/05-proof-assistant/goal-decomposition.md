# Décomposition de buts dans Heaven Core

La décomposition de buts permet :

- de simplifier les preuves,
- de réduire l’espace de recherche,
- d’améliorer l’automatisation.

## Techniques

### 1. Décomposition structurelle
Basée sur la forme du terme.

### 2. Décomposition logique
Séparation des conjonctions, implications, quantificateurs.

### 3. Décomposition dépendante
Indices normalisés avant décomposition.

### 4. Décomposition par contraintes
Propagation logique.

## Visualisation

- arbres de buts,
- logs,
- contextes.
