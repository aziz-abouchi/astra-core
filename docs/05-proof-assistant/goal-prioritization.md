# Priorisation de buts dans Heaven Core

La priorisation de buts permet :

- d’accélérer les preuves,
- de réduire l’espace de recherche,
- d’améliorer l’automatisation.

## Critères

- complexité,
- dépendances,
- contraintes,
- heuristiques.

## Stratégies

### 1. Buts simples d’abord
Réécriture, simplification.

### 2. Buts critiques
Contraintes bloquantes.

### 3. Buts dépendants
Indices, relations.

## Visualisation

- arbres de buts,
- priorités,
- logs.
