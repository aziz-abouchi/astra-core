# Moteur de réécriture dans Astra Core

Le moteur de réécriture est :

- orienté,
- typé,
- extensible.

## Règles

Les règles sont :

- des lemmes `[simp]`,
- des équations,
- des identités.

## Mécanismes

### 1. Réécriture orientée

- direction choisie,
- normalisation.

### 2. Réécriture dépendante

- `subst`,
- `transport`,
- `cong`.

### 3. Réécriture contextuelle

- sous-termes,
- contextes typés.

## Optimisations

- partage,
- mémoïsation,
- normalisation partielle.
