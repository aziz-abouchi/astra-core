# Topologie réseau dans Heaven Core

Heaven Core supporte plusieurs topologies réseau :

- anneau,
- étoile,
- mesh,
- arbre,
- cluster dynamique.

## Détection

Les nœuds détectent :

- leurs voisins,
- les changements,
- les partitions.

## Routage typé

Les messages sont routés selon :

- les protocoles,
- les capacités,
- les budgets.

## Migration

Les fibres peuvent migrer selon la topologie :

- pour réduire la latence,
- pour équilibrer la charge,
- pour rapprocher les données.

## Visualisation

Le runtime expose :

- graphes de topologie,
- métriques par lien,
- heatmaps de latence.
