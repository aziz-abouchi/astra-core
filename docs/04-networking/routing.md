# Routage dans Heaven Core

Le routage est :

- typé,
- frugal,
- dynamique,
- conscient des capacités.

## Types de routage

### 1. Routage direct
Un nœud connaît la destination.

### 2. Routage multi-sauts
Le message traverse plusieurs nœuds.

### 3. Routage par protocole
Les transitions guident le chemin.

### 4. Routage QoS
Selon la priorité ou la latence.

## Algorithmes

- Dijkstra typé,
- gossip,
- anneaux distribués,
- tables de routage incrémentales.

## Migration

Le routage peut déclencher :

- migration de fibres,
- rééquilibrage,
- réplication.

## Visualisation

- graphes de routage,
- heatmaps de latence,
- métriques par lien.
