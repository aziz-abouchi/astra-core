# Monitoring réseau dans Astra Core

Le runtime distribué expose des métriques réseau typées.

## Métriques

- latence,
- bande passante,
- messages envoyés/reçus,
- erreurs de protocole,
- invalidations de capacités.

## Monitoring local

Chaque nœud collecte :

- métriques réseau,
- métriques de protocole,
- métriques de supervision.

## Monitoring distribué

Les superviseurs peuvent :

- agréger les métriques,
- détecter les anomalies,
- déclencher des migrations.

## Visualisation

- graphes de flux,
- heatmaps de latence,
- arbres de protocole.
