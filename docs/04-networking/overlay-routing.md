# Routage overlay dans Astra Core

Les overlays permettent :

- des topologies virtuelles,
- des protocoles spécialisés,
- une fédération flexible.

## Types d’overlays

- anneau,
- mesh,
- arbre,
- DHT,
- cluster fédéré.

## Routage overlay

Le routage overlay est :

- typé,
- supervisé,
- frugal.

## Capacités overlay

Les capacités peuvent être :

- locales,
- distantes,
- fédérées.

## Visualisation

- graphes overlay,
- métriques,
- heatmaps.
