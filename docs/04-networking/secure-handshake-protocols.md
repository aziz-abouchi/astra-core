# Protocoles de handshake sécurisé dans Astra Core

Les handshakes sécurisés garantissent :

- authenticité,
- intégrité,
- confidentialité.

## Étapes

### 1. Négociation de protocole
Versions, états, transitions.

### 2. Négociation de capacités
Permissions, modes, linéarité.

### 3. Établissement sécurisé
Clés, signatures, validation.

## Handshake distribué

Les nœuds peuvent :

- se découvrir,
- se valider,
- se fédérer.
