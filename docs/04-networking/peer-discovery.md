# Découverte de pairs dans Heaven Core

La découverte de pairs permet :

- la formation de clusters,
- la fédération,
- la résilience.

## Mécanismes

### 1. Gossip

Propagation rapide et frugale.

### 2. Broadcast typé

Messages typés pour éviter les collisions.

### 3. Supervision

Les superviseurs détectent :

- les nouveaux nœuds,
- les nœuds défaillants.

## Capacités

Les capacités peuvent être :

- annoncées,
- négociées,
- validées.

## Visualisation

- graphes de pairs,
- heatmaps,
- logs.
