# Edge Computing dans Heaven Core

Heaven Core peut s’exécuter :

- sur des clusters,
- sur des nœuds edge,
- sur des environnements hybrides.

## Objectifs

- faible latence,
- frugalité,
- distribution intelligente.

## Déploiement

Les fibres peuvent être :

- envoyées vers l’edge,
- migrées dynamiquement,
- supervisées à distance.

## Capacités edge

Les capacités peuvent être :

- locales,
- distantes,
- fédérées.

## Protocoles edge

Les protocoles peuvent s’adapter :

- à la latence,
- à la bande passante,
- aux ressources.
