# Réseaux overlay dans Astra Core

Les overlays permettent :

- des topologies virtuelles,
- des protocoles personnalisés,
- des clusters fédérés.

## Types d’overlays

- anneau,
- mesh,
- arbre,
- DHT,
- fédération.

## Routage overlay

Le routage peut être :

- typé,
- QoS,
- supervisé.

## Capacités overlay

Les capacités peuvent être :

- locales,
- distantes,
- fédérées.

## Visualisation

- graphes overlay,
- métriques,
- heatmaps.
