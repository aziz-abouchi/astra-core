# Failover de transport dans Astra Core

Le failover de transport garantit :

- résilience,
- continuité,
- performance.

## Mécanismes

### 1. Détection de panne
Heartbeats typés.

### 2. Reconnexion automatique
Transitions typées.

### 3. Migration de flux
Zero-copy lorsque possible.

### 4. Routage alternatif
QoS, latence, bande passante.

- graphes de failover,
