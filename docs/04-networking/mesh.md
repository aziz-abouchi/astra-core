# Réseau mesh dans Heaven Core

Heaven Core supporte une topologie mesh :

- dynamique,
- résiliente,
- typée.

## Découverte

Les nœuds découvrent :

- leurs voisins,
- les routes,
- les capacités.

## Routage mesh

Le routage mesh utilise :

- Dijkstra typé,
- gossip,
- supervision.

## Avantages

- tolérance aux pannes,
- faible latence,
- scalabilité.

## Visualisation

- graphes mesh,
- heatmaps,
- métriques par lien.
