# Virtualisation réseau dans Astra Core

Astra Core peut fonctionner dans des environnements virtualisés :

- containers,
- VMs,
- sandboxes,
- enclaves.

## Objectifs

- isolation,
- sécurité,
- portabilité.

## Capacités virtualisées

Les capacités peuvent être :

- locales à la VM,
- exposées via RPC,
- fédérées.

## Protocoles virtualisés

Les protocoles peuvent être :

- encapsulés,
- multiplexés,
- routés.

## Migration

Les fibres peuvent migrer entre :

- containers,
- VMs,
- nœuds physiques.
