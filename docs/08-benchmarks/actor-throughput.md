# Benchmarks de throughput des acteurs dans Astra Core

Ces benchmarks mesurent :

- le débit message → message,
- la scalabilité,
- la résilience.

## Tests typiques

### 1. Ping-pong massif

- millions de messages,
- latence,
- jitter.

### 2. Acteurs supervisés

- propagation d’erreurs,
- redémarrages.

### 3. Acteurs distribués

- migration,
- réplication.

## Visualisation

- heatmaps,
- courbes,
