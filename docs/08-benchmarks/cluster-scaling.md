# Benchmarks de scalabilité de cluster dans Heaven Core

Ces benchmarks mesurent :

- la scalabilité horizontale,
- la résilience,
- la performance globale.

## Tests typiques

### 1. Scaling d’acteurs

- millions d’acteurs,
- supervision,
- migration.

### 2. Scaling miniKanren

- branches,
- substitutions,
- distribution.

### 3. Scaling réseau

- RPC,
- streaming,
- consensus.
