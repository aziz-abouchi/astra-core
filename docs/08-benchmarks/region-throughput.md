# Benchmarks de throughput de régions dans Heaven Core

Ces benchmarks mesurent :

- la vitesse d’allocation,
- la vitesse de libération,
- la frugalité.

## Tests typiques

### 1. Allocation massive
Millions d’allocations.

### 2. Régions imbriquées
Profondeur, performance.

### 3. Régions migrées
Sérialisation, transfert.

## Visualisation

- heatmaps,
- courbes,
- timelines.
