# Benchmarks de sérialisation zero-copy dans Heaven Core

Ces benchmarks mesurent :

- la vitesse de sérialisation,
- la vitesse de désérialisation,
- le coût mémoire,
- la frugalité.

## Tests typiques

### 1. Buffers linéaires

- zero-copy,
- transfert direct.

### 2. Protocoles

- transitions,
- messages.

### 3. Migration

- fibres,
- régions,
- capacités.

## Visualisation

- courbes,
- heatmaps,
- comparaisons multi-backends.
