# Mesure d’énergie dans les benchmarks Astra Core

Astra Core peut mesurer la consommation énergétique.

## Objectifs

- frugalité,
- optimisation,
- comparaison multi-langages.

## Méthodes

### 1. Compteurs matériels

- RAPL (Intel),
- PMCs,
- ARM Energy Model.

### 2. Estimation logicielle

- cycles CPU,
- allocations,
- instructions.

### 3. Normalisation

Les résultats sont :

- normalisés par backend,
- comparés entre langages,
- archivés.

## Visualisation

- graphes énergie/temps,
- énergie par fibre,
- énergie par protocole.
