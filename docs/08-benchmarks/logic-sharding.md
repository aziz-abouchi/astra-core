# Benchmarks de sharding logique dans Astra Core

Ces benchmarks mesurent :

- la distribution de la logique,
- la performance de miniKanren,
- la scalabilité.

## Tests typiques

### 1. Sharding de substitutions

- distribution,
- synchronisation.

### 2. Sharding de branches

- parallélisation,
- interleaving.

### 3. Sharding de contraintes

- propagation,
- résolution.

## Visualisation

- arbres distribués,
- heatmaps,
- courbes.
