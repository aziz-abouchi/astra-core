# Benchmarks de sérialisation dans Heaven Core

Ces benchmarks mesurent :

- la vitesse de sérialisation,
- la vitesse de désérialisation,
- la taille des messages,
- la frugalité.

## Tests typiques

### 1. Sérialisation de structures

- listes,
- vecteurs,
- arbres.

### 2. Sérialisation de protocoles

- transitions,
- messages,
- capacités.

### 3. Sérialisation distribuée

- migration de fibres,
- consensus,
- RPC.

## Visualisation

- courbes taille/temps,
- heatmaps,
- comparaisons multi-backends.
