# Benchmarks de throughput de protocoles dans Heaven Core

Ces benchmarks mesurent :

- la vitesse des transitions,
- la latence,
- la frugalité.

## Tests typiques

### 1. Transitions simples
Open → Close → Open.

### 2. Protocoles complexes
États multiples, contraintes.

### 3. Protocoles distribués
Synchronisation inter-nœuds.
