# Benchmarks de throughput dans Heaven Core

Ces benchmarks mesurent :

- le débit,
- la capacité maximale,
- la stabilité sous charge.

## Tests typiques

### 1. Acteurs

- messages par seconde,
- transitions de protocole,
- supervision.

### 2. Pipelines

- éléments par seconde,
- fusion,
- backpressure.

### 3. miniKanren

- substitutions par seconde,
- branches par seconde.

### 4. Réseau

- RPC par seconde,
- streaming,
- consensus.

## Visualisation

- courbes de throughput,
- heatmaps,
- graphes de saturation.
