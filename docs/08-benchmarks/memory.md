# Benchmarks mémoire dans Heaven Core

Ces benchmarks mesurent :

- allocations,
- fragmentation,
- pression mémoire,
- efficacité des arenas,
- efficacité du GC.

## Tests typiques

### 1. Allocation intensive

- listes,
- vecteurs,
- arbres.

### 2. Pipelines

- streaming,
- fusion,
- backpressure.

### 3. miniKanren

- substitutions,
- branches,
- profondeur logique.

## Visualisation

- graphes mémoire/temps,
- heatmaps,
- courbes de fragmentation.
