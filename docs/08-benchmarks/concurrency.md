# Benchmarks de concurrence dans Astra Core

Ces benchmarks mesurent :

- la création de fibres,
- le scheduling,
- la communication,
- la distribution.

## Tests typiques

### 1. Ping-pong

- latence,
- débit,
- transitions de protocole.

### 2. Acteurs

- millions de messages,
- supervision,
- migration.

### 3. miniKanren

- profondeur logique,
- branches par seconde,
- exploration distribuée.

### 4. Pipelines

- streaming,
- backpressure,
- fusion.

## Visualisation

- graphes de charge,
- heatmaps de scheduling,
- courbes de saturation.
