# Benchmarks de persistance dans Astra Core

Ces benchmarks mesurent :

- la vitesse de stockage,
- la sérialisation,
- la réplication,
- la récupération.

## Tests typiques

### 1. Sérialisation massive

- vecteurs,
- arbres,
- protocoles.

### 2. Checkpointing

- fibres,
- régions,
- capacités.

### 3. Réplication

- consensus,
- CRDTs,
- journaux.

## Visualisation

- courbes de persistance,
- heatmaps,
- comparaisons multi-backends.
