# Benchmarks d’acteurs dans Astra Core

Ces benchmarks mesurent :

- la performance du modèle d’acteurs,
- la supervision,
- la distribution,
- la migration.

## Tests typiques

### 1. Ping-pong d’acteurs

- latence,
- débit,
- transitions.

### 2. Acteurs supervisés

- redémarrages,
- propagation d’erreurs,
- budgets.

### 3. Acteurs distribués

- migration,
- réplication,
- consensus.

### 4. Acteurs + miniKanren

- recherche logique dans les acteurs,
- distribution des branches.

## Visualisation

- graphes d’acteurs,
- heatmaps,
- courbes de saturation.
