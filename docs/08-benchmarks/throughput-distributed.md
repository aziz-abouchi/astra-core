# Benchmarks de throughput distribué dans Astra Core

Ces benchmarks mesurent :

- le débit global,
- la scalabilité horizontale,
- la performance inter-nœuds.

## Tests typiques

### 1. Ping-pong distribué

- latence,
- débit,
- transitions.

### 2. Acteurs distribués

- messages par seconde,
- supervision,
- migration.

### 3. miniKanren distribué

- branches par seconde,
- profondeur logique,
- sharding.

### 4. RPC massif

- appels par seconde,
- sérialisation,
- consensus.

## Visualisation

- graphes de throughput,
- heatmaps,
- courbes de saturation.
