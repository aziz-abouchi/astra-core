# Benchmarks de stockage dans Heaven Core

Ces benchmarks mesurent :

- la vitesse d’écriture,
- la vitesse de lecture,
- la sérialisation,
- la réplication.

## Tests typiques

### 1. Sérialisation massive

- vecteurs,
- arbres,
- protocoles.

### 2. Checkpointing

- fibres,
- régions,
- capacités.

### 3. Réplication

- consensus,
- CRDTs,
- journaux.

## Visualisation

- courbes,
- heatmaps,
- comparaisons multi-backends.
