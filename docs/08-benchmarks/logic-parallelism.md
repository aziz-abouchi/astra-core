# Benchmarks de parallélisme logique dans Heaven Core

Ces benchmarks mesurent :

- la parallélisation de miniKanren,
- la distribution des branches,
- la scalabilité.

## Tests typiques

### 1. Branches parallèles

- interleaving,
- profondeur,
- substitutions.

### 2. Sharding logique

- distribution,
- synchronisation.

### 3. Logique + acteurs

- recherche distribuée,
- supervision.
