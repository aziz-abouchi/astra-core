# Benchmarks de latence des acteurs dans Astra Core

Ces benchmarks mesurent :

- la latence message → traitement,
- la latence message → message,
- la latence de supervision.

## Tests typiques

### 1. Ping-pong d’acteurs

- latence,
- jitter,
- transitions.

### 2. Acteurs supervisés

- redémarrages,
- propagation d’erreurs.

### 3. Acteurs distribués

- migration,
- réplication.
