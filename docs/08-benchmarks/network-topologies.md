# Benchmarks de topologies réseau dans Astra Core

Ces benchmarks mesurent :

- la performance selon la topologie,
- la résilience,
- la scalabilité.

## Topologies testées

- anneau,
- étoile,
- mesh,
- arbre,
- cluster fédéré.

## Tests typiques

### 1. Ping-pong

- latence,
- jitter.

### 2. RPC massif

- throughput,
- saturation.

### 3. Consensus

- stabilité,
- performance.

## Visualisation

- graphes,
- heatmaps,
- courbes.
