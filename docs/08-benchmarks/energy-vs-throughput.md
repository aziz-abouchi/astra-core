# Benchmarks énergie vs throughput dans Heaven Core

Ces benchmarks mesurent :

- la consommation énergétique,
- le throughput,
- la frugalité.

## Tests typiques

### 1. Acteurs

- messages par joule,
- latence,
- supervision.

### 2. miniKanren

- substitutions par joule,
- profondeur logique.

### 3. RPC

- appels par joule,
- sérialisation.

## Visualisation

- graphes énergie/throughput,
- heatmaps,
- comparaisons multi-backends.
