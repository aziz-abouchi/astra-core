# Benchmarks de logique dans Astra Core

Ces benchmarks mesurent :

- la performance de miniKanren,
- la profondeur logique,
- la vitesse d’unification,
- la distribution logique.

## Tests typiques

### 1. Unification intensive

- substitutions,
- variables fraîches,
- contraintes.

### 2. `appendo`

- profondeur,
- branches,
- solutions.

### 3. Recherche distribuée

- sharding logique,
- interleaving,
- budgets.

### 4. Preuves + logique

- interaction entre preuves et goals,
- contraintes dépendantes.

## Visualisation

- arbres de recherche,
- heatmaps,
- courbes de profondeur.
