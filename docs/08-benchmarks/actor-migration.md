# Benchmarks de migration d’acteurs dans Heaven Core

Ces benchmarks mesurent :

- la vitesse de migration,
- la latence,
- la frugalité,
- la résilience.

## Tests typiques

### 1. Migration simple

- état minimal,
- latence.

### 2. Migration massive

- milliers d’acteurs,
- supervision.

### 3. Migration + logique

- substitutions,
- contraintes.
