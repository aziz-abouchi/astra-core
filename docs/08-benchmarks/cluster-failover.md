# Benchmarks de failover de cluster dans Astra Core

Ces benchmarks mesurent :

- la résilience,
- la vitesse de reprise,
- la stabilité.

## Tests typiques

### 1. Crash de nœud

- redémarrage,
- migration,
- supervision.

### 2. Partition réseau

- consensus,
- réplication.

### 3. Surcharge

- saturation,
- backpressure.
