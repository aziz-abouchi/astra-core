# Scalabilité dans les benchmarks Heaven Core

Les benchmarks de scalabilité mesurent :

- la croissance du temps,
- la croissance mémoire,
- la croissance énergétique,
- la croissance du nombre de fibres.

## Types de tests

### 1. Scalabilité verticale

- taille des données,
- profondeur logique,
- taille des vecteurs.

### 2. Scalabilité horizontale

- nombre de nœuds,
- nombre d’acteurs,
- nombre de branches miniKanren.

### 3. Scalabilité réseau

- latence,
- bande passante,
- saturation.

## Visualisation

- graphes log-log,
- heatmaps,
- courbes de saturation.
