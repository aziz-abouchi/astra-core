# Automatisation des benchmarks

Astra Core fournit un framework d’automatisation des benchmarks.

## Objectifs

- reproductibilité,
- multi-backends,
- multi-langages,
- CI/CD.

## Scripts

Les scripts :

- compilent Astra, Roc, Zig, Bun, PHP, Lys,
- exécutent les benchmarks,
- collectent les métriques,
- génèrent des rapports.

## Intégration CI

- GitHub Actions,
- runners multi-OS,
- artefacts reproductibles.

## Résultats

Les résultats sont :

- archivés,
- comparés,
- visualisés (graphes, tableaux).
