# Benchmarks de pipeline de sérialisation dans Astra Core

Ces benchmarks mesurent :

- la vitesse de sérialisation,
- la vitesse de désérialisation,
- la frugalité,
- la scalabilité.

## Pipeline

1. allocation,
2. sérialisation,
3. compression,
4. transport,
5. désérialisation.

## Tests typiques

- buffers linéaires,
- protocoles,
- régions,
- migrations.

## Visualisation

- courbes,
- heatmaps,
- comparaisons.
