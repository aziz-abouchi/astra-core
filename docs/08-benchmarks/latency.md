# Mesure de latence dans Heaven Core

La latence est une métrique clé pour :

- le runtime distribué,
- les protocoles,
- les RPC.

## Méthodes

### 1. Latence locale

- mesure du temps entre `send` et `recv`,
- mesure des transitions de protocole.

### 2. Latence réseau

- round-trip time,
- latence par nœud,
- latence par protocole.

### 3. Latence logique

- profondeur de recherche miniKanren,
- temps par branche,
- temps par substitution.

## Visualisation

- histogrammes,
- heatmaps,
- graphes par protocole.
