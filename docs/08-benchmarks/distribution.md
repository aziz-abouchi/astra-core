# Benchmarks distribués dans Astra Core

Ces benchmarks mesurent :

- la performance du scheduler distribué,
- la latence inter-nœuds,
- la migration de fibres,
- la réplication.

## Tests typiques

### 1. Ping-pong distribué

- latence,
- débit,
- transitions de protocole.

### 2. Migration de fibres

- coût de migration,
- sérialisation,
- réhydratation.

### 3. miniKanren distribué

- branches par nœud,
- profondeur logique,
- exploration parallèle.

### 4. Consensus

- latence,
- throughput,
- stabilité.

## Visualisation

- graphes de cluster,
- heatmaps de latence,
- courbes de saturation.
