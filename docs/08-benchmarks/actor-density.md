# Benchmarks de densité d’acteurs dans Heaven Core

Ces benchmarks mesurent :

- le nombre maximal d’acteurs,
- la densité mémoire,
- la frugalité.

## Tests typiques

### 1. Densité locale
Millions d’acteurs sur un nœud.

### 2. Densité distribuée
Acteurs répartis sur un cluster.

### 3. Densité + logique
Acteurs + miniKanren.
