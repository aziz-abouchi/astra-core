# Scheduling distribué par priorité dans Heaven Core

Le scheduler distribué gère :

- les priorités,
- les budgets,
- les migrations,
- la frugalité.

## Priorités

- `Low`
- `Normal`
- `High`
- `Critical`

## Scheduling inter-nœuds

Les fibres peuvent être :

- migrées selon la priorité,
- rééquilibrées,
- supervisées.

## Budgets distribués

Chaque fibre possède :

- un budget CPU,
- un budget mémoire,
- un budget énergie.

## Visualisation

- timelines,
- heatmaps,
- graphes de charge.
