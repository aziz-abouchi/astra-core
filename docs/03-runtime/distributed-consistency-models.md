# Modèles de cohérence distribuée dans Astra Core

Astra Core supporte plusieurs modèles de cohérence :

- cohérence causale,
- cohérence séquentielle,
- cohérence par CRDT,
- cohérence supervisée.

## Cohérence causale

Les messages respectent :

- l’ordre logique,
- les dépendances.

## Cohérence séquentielle

Utilisée pour :

- consensus,
- ledger distribué.

## CRDTs

Les structures convergent automatiquement :

- sets,
- maps,
- registres.

## Cohérence supervisée

Les superviseurs garantissent :

- absence de divergence,
- reprise cohérente,
- migration sûre.

## Visualisation

- graphes de cohérence,
- timelines,
- heatmaps.
