# Acteurs distribués avancés dans Heaven Core

Les acteurs distribués avancés supportent :

- migration intelligente,
- réplication,
- supervision multi-nœuds,
- frugalité.

## Migration intelligente

Basée sur :

- latence,
- charge,
- énergie,
- proximité des données.

## Réplication

Les acteurs critiques peuvent être :

- répliqués,
- synchronisés,
- supervisés.

## Acteurs + logique

Les acteurs peuvent exécuter des goals miniKanren distribués.
