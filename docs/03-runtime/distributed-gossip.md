# Gossip distribué dans Astra Core

Le gossip distribué permet :

- la découverte de pairs,
- la propagation d’état,
- la résilience.

## Propriétés

- frugalité,
- convergence,
- supervision.

## Types de messages gossip

- état des nœuds,
- capacités disponibles,
- métriques,
- supervision.

## Gossip typé

Les messages gossip sont typés pour éviter :

- les collisions,
- les incohérences,
- les divergences.

## Visualisation

- graphes de propagation,
- heatmaps,
- timelines.
