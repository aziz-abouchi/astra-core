# Snapshots distribués avancés dans Heaven Core

Les snapshots avancés permettent :

- la reprise exacte,
- la simulation,
- le debugging profond,
- la migration globale.

## Caractéristiques

- cohérence globale,
- compression,
- sérialisation typée,
- supervision.

## Snapshots incrémentaux

Seules les différences sont capturées :

- régions modifiées,
- fibres actives,
- capacités mises à jour.

## Snapshots déterministes

En mode test :

- ordre fixe,
- transitions contrôlées,
- reproduction exacte.

## Visualisation

- graphes de snapshot,
- timelines,
- heatmaps.
