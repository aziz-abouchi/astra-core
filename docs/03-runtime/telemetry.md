# Télémétrie du runtime Astra Core

La télémétrie fournit une vision globale du système.

## Données collectées

- métriques CPU,
- mémoire,
- réseau,
- fibres,
- logique,
- distribution.

## Export

La télémétrie peut être exportée :

- en JSON,
- en flux continu,
- vers un dashboard.

## Télémétrie distribuée

Chaque nœud :

- collecte ses métriques,
- les agrège,
- les transmet au superviseur global.

## Visualisation

- graphes,
- heatmaps,
- timelines.
