# Gestion de cluster dans Heaven Core

Le runtime distribué gère :

- les nœuds,
- les fibres,
- les capacités,
- les protocoles,
- les migrations.

## Découverte

Les nœuds se découvrent via :

- gossip,
- annonces typées,
- supervision.

## Gestion des nœuds

- ajout,
- retrait,
- redémarrage,
- supervision.

## Migration

Les fibres peuvent être migrées :

- pour équilibrer la charge,
- pour réduire la latence,
- pour rapprocher les données.

## Réplication

Les services peuvent être :

- répliqués,
- supervisés,
- synchronisés.

## Visualisation

- graphes de cluster,
- heatmaps,
- métriques.
