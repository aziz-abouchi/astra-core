# Tracing distribué dans Astra Core

Le tracing distribué permet :

- de suivre les fibres,
- de suivre les messages,
- de suivre les transitions de protocole,
- de suivre les migrations.

## Modèle

Chaque nœud :

- capture ses traces,
- les synchronise,
- les exporte.

## Types de traces

- scheduling,
- messages,
- consensus,
- logique,
- capacités.

## Visualisation

- timelines,
- graphes,
- heatmaps.

## Intégration

- debugging,
- simulation,
- tests distribués.
