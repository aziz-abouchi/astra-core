# Ledger distribué dans Astra Core

Astra Core inclut un ledger distribué typé pour :

- la réplication,
- la coordination,
- la persistance.

## Modèle

Le ledger est :

- append-only,
- typé,
- supervisé.

## Entrées

Chaque entrée est :

- sérialisée,
- signée,
- validée.

## Consensus

Le ledger utilise :

- Raft,
- Multi-Paxos,
- CRDTs.

## Utilisations

- audit,
- réplication,
- coordination,
- transactions.
