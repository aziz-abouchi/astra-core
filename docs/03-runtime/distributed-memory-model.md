# Modèle mémoire distribué dans Heaven Core

Le modèle mémoire distribué garantit :

- cohérence,
- frugalité,
- sécurité.

## Propriétés

### 1. Pas de mémoire partagée
Tout est message-passing.

### 2. Régions distribuées
Les régions peuvent être :

- migrées,
- sérialisées,
- synchronisées.

### 3. Buffers linéaires
Zero-copy local et distribué.

### 4. Capacités mémoire
Contrôle de l’accès.

## Visualisation

- graphes mémoire,
- heatmaps,
- timelines.
