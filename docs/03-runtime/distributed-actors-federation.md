# Fédération d’acteurs distribués dans Heaven Core

La fédération permet :

- de connecter plusieurs clusters,
- de partager des acteurs,
- de synchroniser des protocoles.

## Propriétés

- isolation,
- supervision,
- frugalité.

## Fédération typée

Les acteurs fédérés respectent :

- les protocoles,
- les capacités,
- les régions.

## Migration inter‑fédération

Les acteurs peuvent migrer entre clusters :

- latence optimisée,
- sécurité,
- supervision.

## Visualisation

- graphes de fédération,
- timelines,
- heatmaps.
