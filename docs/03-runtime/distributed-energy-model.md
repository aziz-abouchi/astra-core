# Modèle énergétique distribué dans Astra Core

Le modèle énergétique distribué permet :

- d’optimiser la frugalité,
- de mesurer la consommation,
- de guider le scheduling.

## Métriques

- énergie par fibre,
- énergie par protocole,
- énergie par nœud,
- énergie par cluster.

## Budgets distribués

Chaque fibre possède :

- un budget local,
- un budget global,
- un budget par protocole.

## Optimisations

- migration vers nœuds moins coûteux,
- fusion de fibres,
- réduction des allocations.

## Visualisation

- graphes énergie/temps,
- heatmaps,
- comparaisons multi-nœuds.
