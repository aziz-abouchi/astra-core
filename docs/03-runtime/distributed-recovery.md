# Mécanismes de reprise distribuée dans Astra Core

La reprise distribuée garantit :

- la résilience,
- la continuité,
- la cohérence.

## Types de reprise

### 1. Reprise locale
Redémarrage d’acteurs.

### 2. Reprise régionale
Restauration de régions.

### 3. Reprise globale
Snapshots + consensus.

## Reprise typée

Les transitions de reprise sont vérifiées :

- pas d’état impossible,
- pas de capacité invalide,
- pas de fuite.
