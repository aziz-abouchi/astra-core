# Hot Reload dans Heaven Core

Heaven Core supporte le hot reload :

- des modules,
- des acteurs,
- des protocoles,
- des fibres.

## Objectifs

- développement rapide,
- correction à chaud,
- mise à jour continue.

## Mécanismes

### 1. Rechargement de module

- recompilation,
- invalidation du cache,
- migration des fibres.

### 2. Mise à jour d’acteur

- état préservé,
- protocole vérifié,
- supervision.

### 3. Mise à jour distribuée

- propagation,
- validation,
- rollback.

## Sécurité

- pas de changement incompatible,
- vérification typée,
- invariants préservés.
