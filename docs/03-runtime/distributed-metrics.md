# Métriques distribuées dans Heaven Core

Les métriques distribuées permettent :

- d’observer le cluster,
- d’optimiser la frugalité,
- de diagnostiquer les problèmes.

## Types de métriques

- CPU,
- mémoire,
- réseau,
- énergie,
- logique,
- acteurs,
- consensus.

## Agrégation

Chaque nœud :

- collecte localement,
- agrège,
- transmet au superviseur global.

## Visualisation

- dashboards,
- heatmaps,
- timelines.
