# Sémantique de la logique distribuée dans Heaven Core

La logique distribuée combine :

- miniKanren,
- scheduling distribué,
- sharding logique,
- contraintes synchronisées.

## Sharding logique

Les branches logiques peuvent être :

- réparties entre nœuds,
- migrées,
- fusionnées.

## Substitutions distribuées

Les substitutions sont :

- typées,
- compressées,
- synchronisées.

## Contraintes distribuées

Les contraintes peuvent être :

- locales,
- globales,
- propagées via gossip.

## Scheduling logique distribué

Le scheduler :

- intercale les branches,
- respecte les budgets,
- synchronise les résultats.

## Visualisation

- arbres distribués,
- heatmaps,
- timelines.
