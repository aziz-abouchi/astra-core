# Visualiseur de cluster Heaven Core

Le visualiseur permet :

- d’inspecter le cluster,
- de visualiser les migrations,
- de diagnostiquer les problèmes.

## Fonctionnalités

- graphes d’acteurs,
- graphes de régions,
- graphes de capacités,
- heatmaps.

## Visualisation

- temps réel,
- historique,
- comparatif.
