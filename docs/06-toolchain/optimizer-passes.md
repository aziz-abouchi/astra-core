# Passes d’optimisation dans Heaven Core

Le compilateur applique plusieurs passes :

## Passes principales

### 1. Inlining

- fonctions,
- dictionnaires,
- preuves.

### 2. Fusion

- listes,
- vecteurs,
- flux.

### 3. Élimination des preuves

- erasure,
- simplification.

### 4. Optimisation des régions

- arènes,
- libération groupée.

### 5. Optimisation réseau

- sérialisation,
- batching,
- zero-copy.

## Visualisation

- graphes IR,
- statistiques,
- logs.
