# REPL Astra Core

Le REPL est un environnement interactif :

- typage en direct,
- exécution de fibres,
- exploration logique,
- visualisation de preuves.

## Fonctionnalités

- chargement de modules,
- inspection de types,
- exécution de goals miniKanren,
- affichage des substitutions,
- affichage des preuves (avant erasure),
- mode déterministe pour les tests.

## Commandes

- `:type expr`
- `:load module`
- `:prove goal`
- `:logic goal`
- `:resources`
