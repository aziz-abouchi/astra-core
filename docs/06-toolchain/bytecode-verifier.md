# Vérificateur de bytecode Heaven Core

Le vérificateur garantit :

- la sûreté,
- la cohérence,
- l’absence d’UB.

## Vérifications

### 1. Types
Correspondance entre instructions et types.

### 2. Capacités
Pas de duplication, pas de fuite.

### 3. Protocoles
Transitions valides.

### 4. Régions
Pas d’accès hors région.
