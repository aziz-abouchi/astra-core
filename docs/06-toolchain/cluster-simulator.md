# Simulateur de cluster Astra Core

Le simulateur permet :

- de tester sans cluster réel,
- de simuler la latence,
- de simuler les pannes,
- de valider les protocoles.

## Fonctionnalités

- topologies virtuelles,
- latence configurable,
- chaos testing,
- supervision.

## Visualisation

- graphes,
- timelines,
- heatmaps.
