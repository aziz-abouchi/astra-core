# Débogueur de cluster Heaven Core

Le débogueur distribué permet :

- d’inspecter les fibres,
- de suivre les migrations,
- de visualiser les protocoles.

## Fonctionnalités

- breakpoints distribués,
- inspection d’acteurs,
- inspection de régions,
- inspection de capacités.

## Visualisation

- timelines,
- graphes,
- heatmaps.
