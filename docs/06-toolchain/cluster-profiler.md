# Profiler de cluster Heaven Core

Le profiler distribué permet :

- d’analyser la performance,
- de détecter les goulots,
- d’optimiser la frugalité.

## Métriques

- CPU,
- mémoire,
- réseau,
- énergie,
- logique,
- acteurs.

## Fonctionnalités

- profils par nœud,
- profils par protocole,
- profils par fibre.

## Visualisation

- heatmaps,
- graphes,
- timelines.
