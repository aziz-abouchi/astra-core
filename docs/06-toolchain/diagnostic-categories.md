# Catégories de diagnostics dans Astra Core

Les diagnostics sont classés pour faciliter :

- la compréhension,
- la correction,
- l’automatisation.

## Catégories

### 1. Typage

- mismatch,
- ambiguïtés,
- contraintes non résolues.

### 2. Preuves

- obligations non prouvées,
- tactiques échouées.

### 3. Protocoles

- transitions invalides,
- états impossibles.

### 4. Capacités

- duplication,
- fuite,
- invalidation.

### 5. Distribution

- pannes,
- latence,
- incohérences.

## Formats

- texte,
- JSON,
- LSP.
