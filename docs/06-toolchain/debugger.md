# Debugger Astra Core

Le debugger d’Astra Core est conçu pour :

- la concurrence,
- la logique,
- les preuves,
- la distribution.

## Fonctionnalités

- inspection de fibres,
- visualisation des substitutions logiques,
- suivi des budgets,
- inspection des capacités,
- mode déterministe pour la reproductibilité.

## Points d’arrêt

- sur une fibre,
- sur un goal logique,
- sur une transition de protocole,
- sur une étape de preuve (avant erasure).

## Visualisation

Le debugger peut afficher :

- graphes de fibres,
- graphes de protocole,
- arbres de recherche logique,
- traces de scheduling.
