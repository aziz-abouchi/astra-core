# Analyseur de cluster Heaven Core

L’analyseur distribué permet :

- d’inspecter le cluster,
- d’optimiser la frugalité,
- de diagnostiquer les problèmes.

## Fonctionnalités

- analyse des acteurs,
- analyse des régions,
- analyse des protocoles,
- analyse des capacités.

## Visualisation

- heatmaps,
- timelines,
- graphes.
