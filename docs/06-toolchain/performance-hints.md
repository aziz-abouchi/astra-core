# Performance Hints dans Astra Core

Le toolchain fournit des conseils de performance.

## Types de hints

### 1. Allocation

- trop d’allocations,
- pas de fusion,
- pas d’arena.

### 2. Linéarité

- ressources non libérées,
- transferts inutiles.

### 3. Protocoles

- transitions inutiles,
- sérialisation coûteuse.

### 4. Logique

- unification lente,
- branches profondes.

## Formats

- LSP,
- CLI,
- rapports.
