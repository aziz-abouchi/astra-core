# Optimiseur de bytecode Heaven Core

L’optimiseur de bytecode applique :

- fusion,
- inlining,
- élimination de preuves,
- zero-copy,
- arènes.

## Passes principales

### 1. Simplification
Réduction des instructions.

### 2. Fusion
Combinaison de séquences.

### 3. Allocation optimisée
Arènes, régions.

### 4. Optimisation réseau
Batching, compression.
