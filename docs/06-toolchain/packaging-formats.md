# Formats de packaging dans Heaven Core

Heaven Core supporte plusieurs formats de packaging.

## Formats

### 1. `.heaven`
Format source.

### 2. `.acore`
IR compilé.

### 3. `.apkg`
Paquet compilé + métadonnées.

### 4. `.adoc`
Documentation générée.

## Métadonnées

- version,
- dépendances,
- preuves vérifiées,
- protocoles.

## Distribution

Les paquets peuvent être :

- locaux,
- distants,
- fédérés.
