# Profiler Heaven Core

Le profiler mesure :

- temps CPU,
- allocations,
- consommation mémoire,
- nombre de fibres,
- profondeur de recherche logique.

## Modes

- **Sampling** : léger, continu.
- **Tracing** : détaillé, coûteux.
- **Logic profiling** : suivi des branches miniKanren.

## Visualisation

Le profiler peut afficher :

- flamegraphs,
- graphes de fibres,
- arbres de recherche,
- budgets consommés.

## Intégration avec le runtime

Le profiler peut :

- déclencher des alertes,
- ajuster les priorités,
- informer les superviseurs.
