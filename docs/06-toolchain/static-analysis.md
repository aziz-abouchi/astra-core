# Analyse statique dans Astra Core

Le toolchain inclut un analyseur statique typé.

## Objectifs

- détecter les erreurs,
- optimiser,
- vérifier les invariants,
- analyser la frugalité.

## Analyses

### 1. Analyse d’ownership

- ressources linéaires,
- capacités,
- transferts.

### 2. Analyse d’effets

- IO,
- logique,
- distribution.

### 3. Analyse de protocoles

- transitions,
- états,
- erreurs.

### 4. Analyse mémoire

- allocations,
- fuites,
- fragmentation.

## Intégration

- LSP,
- CI,
- optimiseur.
