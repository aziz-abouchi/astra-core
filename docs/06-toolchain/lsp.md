# LSP (Language Server Protocol) pour Astra Core

Astra Core fournit un serveur LSP complet.

## Fonctionnalités

- complétion,
- navigation,
- documentation,
- inspection de types,
- affichage des preuves,
- visualisation des protocoles,
- diagnostics.

## Intégration

Compatible avec :

- VS Code,
- Neovim,
- Emacs,
- JetBrains (via plugin).

## Diagnostics

- erreurs de typage,
- erreurs de preuve,
- erreurs de protocole,
- erreurs de capacités.

## Visualisation

- graphes de types,
- graphes de protocoles,
- arbres de preuves.
