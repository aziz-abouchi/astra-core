# Visualiseur IR dans Astra Core

Le visualiseur IR permet :

- d’inspecter le code compilé,
- de comprendre les optimisations,
- de diagnostiquer les problèmes.

## Fonctionnalités

- graphes IR,
- affichage des passes,
- comparaison avant/après.

## Formats

- HTML,
- JSON,
- SVG.

## Intégration

- LSP,
- CI,
- CLI.
