# Diagnostics du toolchain Heaven Core

Le toolchain fournit des diagnostics :

- précis,
- typés,
- contextualisés.

## Types de diagnostics

### 1. Erreurs de typage

- mismatch,
- contraintes,
- indices.

### 2. Erreurs de preuve

- obligations non prouvées,
- réécriture invalide.

### 3. Erreurs de protocole

- transitions invalides,
- états impossibles.

### 4. Erreurs de capacités

- capacité manquante,
- capacité invalide.

## Formats

- texte,
- JSON,
- LSP.

## Visualisation

- graphes,
- contextes,
- traces.
