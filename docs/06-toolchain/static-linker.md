# Linker statique dans Heaven Core

Le linker statique permet :

- des binaires autonomes,
- une distribution simple,
- une frugalité maximale.

## Fonctionnalités

- élimination de code mort,
- fusion des modules,
- compression,
- vérification des capacités.

## Sorties

- binaire natif,
- bytecode optimisé,
- WASM statique.

## Visualisation

- graphes de dépendances,
- statistiques,
- logs.
