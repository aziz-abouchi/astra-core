# Bytecode Astra Core

Astra Core compile vers un bytecode portable.

## Objectifs

- portabilité,
- frugalité,
- optimisation,
- distribution.

## Structure

Le bytecode contient :

- instructions,
- métadonnées,
- informations de protocole,
- informations de capacité.

## Exécution

Le bytecode est exécuté par :

- l’interpréteur,
- le JIT,
- le runtime distribué.

## Optimisations

- fusion,
- inlining,
- zero-copy,
- arènes.
