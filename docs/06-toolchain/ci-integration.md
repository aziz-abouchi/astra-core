# Intégration CI pour Heaven Core

Heaven Core s’intègre facilement dans les pipelines CI/CD.

## Objectifs

- reproductibilité,
- vérification des preuves,
- benchmarks automatiques,
- tests distribués.

## Étapes typiques

1. build,
2. vérification des preuves,
3. tests unitaires,
4. tests distribués,
5. fuzzing,
6. benchmarks,
7. génération de docs.

## Artefacts

- binaires,
- rapports de tests,
- rapports de couverture,
- documentation.

## Intégration

- GitHub Actions,
- GitLab CI,
- Azure Pipelines.
