# Link-Time Optimization (LTO) de cluster dans Heaven Core

La LTO distribuée permet :

- fusion inter-nœuds,
- élimination de code mort,
- frugalité maximale.

## Fonctionnement

- hashing global,
- fusion de bytecode,
- optimisation croisée.

## Visualisation

- graphes IR,
- statistiques,
- logs.
