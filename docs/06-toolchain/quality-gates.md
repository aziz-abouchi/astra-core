# Quality Gates dans Astra Core

Les quality gates garantissent :

- la qualité,
- la stabilité,
- la sûreté.

## Gates disponibles

### 1. Typage

- pas d’erreurs,
- pas d’ambiguïtés.

### 2. Preuves

- toutes les obligations prouvées,
- pas de tactiques non terminantes.

### 3. Protocoles

- transitions valides,
- états cohérents.

### 4. Capacités

- pas de fuite,
- pas de duplication.

### 5. Benchmarks

- performance minimale,
- frugalité.

## Intégration

- CI/CD,
- LSP,
- packaging.
