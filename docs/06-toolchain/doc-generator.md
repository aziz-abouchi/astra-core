# Générateur de documentation Heaven Core

Heaven Core inclut un générateur de documentation typée.

## Fonctionnalités

- extraction automatique des signatures,
- extraction des preuves,
- extraction des protocoles,
- génération de graphes.

## Formats

- HTML,
- Markdown,
- JSON (pour outils externes).

## Documentation interactive

- visualisation des protocoles,
- visualisation des preuves,
- navigation dans les modules.

## Intégration

- LSP,
- CI,
- site statique.
