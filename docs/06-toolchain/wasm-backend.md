# Backend WebAssembly (WASM) dans Heaven Core

Heaven Core peut compiler vers WASM pour :

- le web,
- les environnements embarqués,
- les sandboxes.

## Objectifs

- portabilité,
- sécurité,
- frugalité.

## Fonctionnalités

- compilation du bytecode vers WASM,
- runtime minimal,
- capacités simulées,
- IO restreints.

## Limitations

- pas de threads natifs,
- capacités limitées,
- pas de FFI complet.

## Visualisation

- graphes IR,
- statistiques,
- logs.
