# Couverture de tests dans Astra Core

Astra Core mesure la couverture :

- des branches,
- des patterns,
- des protocoles,
- des preuves.

## Types de couverture

### 1. Couverture de code

- lignes,
- branches,
- expressions.

### 2. Couverture logique

- branches miniKanren explorées,
- substitutions générées.

### 3. Couverture de protocoles

- transitions couvertes,
- états atteints.

### 4. Couverture de preuves

- lemmes utilisés,
- tactiques utilisées.

## Visualisation

- heatmaps,
- graphes,
- rapports HTML.
