# Energy Fuzzing dans Heaven Core

Le energy fuzzing permet :

- de tester la frugalité,
- de détecter les pics énergétiques,
- d’optimiser les algorithmes.

## Générateurs

- charges,
- topologies,
- protocoles,
- acteurs.

## Métriques

- énergie par opération,
- énergie par message,
- énergie par substitution.

## Visualisation

- courbes énergie/temps,
- heatmaps,
- comparaisons.
