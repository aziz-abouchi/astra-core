# Stress tests de régions dans Astra Core

Les stress tests de régions mesurent :

- la performance,
- la fragmentation,
- la frugalité.

## Tests typiques

### 1. Allocation massive

- millions d’allocations,
- arènes,
- libération groupée.

### 2. Régions imbriquées

- profondeur,
- performance.

### 3. Régions migrées

- sérialisation,
- transfert,
- supervision.

## Visualisation

- heatmaps,
- courbes,
- timelines.
