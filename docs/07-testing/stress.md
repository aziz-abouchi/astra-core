# Stress Testing dans Heaven Core

Les stress tests mesurent :

- la robustesse,
- la stabilité,
- la tolérance aux pannes.

## Types de stress tests

### 1. Charge extrême

- millions de fibres,
- millions de messages,
- pipelines saturés.

### 2. miniKanren extrême

- profondeur logique,
- branches massives,
- budgets serrés.

### 3. Réseau extrême

- latence variable,
- perte de paquets,
- partitions.

### 4. Distribution extrême

- migration massive,
- consensus intensif,
- réplication.

## Visualisation

- graphes de saturation,
- heatmaps,
- courbes de stabilité.
