# Tests de cohérence dans Astra Core

Les tests de cohérence garantissent :

- l’absence de divergence,
- la stabilité des invariants,
- la cohérence des protocoles.

## Types de cohérence

### 1. Cohérence logique

- substitutions,
- branches,
- contraintes.

### 2. Cohérence mémoire

- ownership,
- linéarité,
- régions.

### 3. Cohérence réseau

- transitions,
- états,
- capacités.

## Outils

- trace replay,
- simulation,
- chaos testing.
