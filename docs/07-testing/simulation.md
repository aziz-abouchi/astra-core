# Simulation dans Astra Core

Astra Core permet de simuler :

- des clusters,
- des protocoles,
- des pannes,
- des budgets.

## Objectifs

- tests déterministes,
- validation de protocoles,
- simulation de charge,
- simulation de réseau.

## Mécanismes

### 1. Cluster simulé

- plusieurs nœuds virtuels,
- scheduler distribué déterministe,
- canaux simulés.

### 2. Simulation de pannes

- crash de nœud,
- perte de messages,
- partitions réseau.

### 3. Simulation de charge

- fibres massives,
- branches logiques,
- IO intensifs.

## Traces

Les simulations produisent :

- logs typés,
- traces de scheduling,
- métriques.
