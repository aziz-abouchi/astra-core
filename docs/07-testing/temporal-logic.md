# Logique temporelle dans Heaven Core

La logique temporelle permet :

- de vérifier des propriétés dynamiques,
- de raisonner sur les transitions,
- de valider les protocoles.

## Opérateurs

### 1. `G` — toujours

### 2. `F` — éventuellement

### 3. `X` — prochain état

### 4. `U` — jusqu’à

## Utilisations

- protocoles,
- consensus,
- supervision,
- scheduling.

## Intégration

- model checking,
- simulation,
- tests distribués.
