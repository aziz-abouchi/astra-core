# Logic Chaos Testing dans Heaven Core

Le logic chaos testing permet :

- de tester la robustesse logique,
- de détecter les divergences,
- de valider les budgets.

## Types de chaos logique

- explosion de branches,
- substitutions massives,
- contraintes contradictoires,
- profondeur extrême.

## Visualisation

- arbres distribués,
- heatmaps,
- logs.
