# Region Fault Testing dans Astra Core

Le region fault testing permet :

- de tester la robustesse mémoire,
- de détecter les fuites,
- de valider les invariants.

## Types de fautes

- fragmentation,
- corruption,
- surcharge,
- migration rapide.

## Visualisation

- heatmaps,
- timelines,
- métriques.
