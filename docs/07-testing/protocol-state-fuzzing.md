# Protocol State Fuzzing dans Heaven Core

Le fuzzing d’états de protocole permet :

- de tester les transitions,
- de détecter les états impossibles,
- de valider la cohérence.

## Générateurs

- états,
- transitions,
- messages.

## Scénarios

### 1. Transitions invalides
Détection automatique.

### 2. États impossibles
Propagation de contraintes.

### 3. Protocoles distribués
Synchronisation multi-nœuds.

## Visualisation

- graphes d’états,
- heatmaps,
- logs.
