# Protocol Chaos Testing dans Astra Core

Le chaos testing de protocoles permet :

- de tester les transitions,
- de détecter les états impossibles,
- de valider la résilience.

## Types de chaos

- transitions invalides,
- messages corrompus,
- latence extrême,
- partitions.

## Visualisation

- graphes d’états,
- logs,
- heatmaps.
