# Tests de régression dans Heaven Core

Les tests de régression garantissent :

- la stabilité,
- la compatibilité,
- la reproductibilité.

## Types de tests

### 1. Tests de comportement

- fonctions,
- protocoles,
- acteurs.

### 2. Tests logiques

- substitutions miniKanren,
- exploration déterministe.

### 3. Tests de preuves

- normalisation,
- obligations,
- extraction.

## Snapshots typés

Les snapshots sont :

- sérialisés,
- typés,
- reproductibles.

## Intégration CI

- comparaison automatique,
- détection des régressions,
- rapports détaillés.
