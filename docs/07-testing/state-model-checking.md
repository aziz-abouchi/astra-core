# Model Checking d’état dans Astra Core

Le model checking vérifie :

- les protocoles,
- les transitions,
- les invariants.

## Modèle

Les états sont :

- typés,
- finis ou infinis,
- explorés symboliquement.

## Propriétés

- sûreté,
- vivacité,
- absence de deadlocks.

## Intégration

- simulation,
- fuzzing,
- tests distribués.

## Visualisation

- graphes d’états,
- contre-exemples,
- traces.
