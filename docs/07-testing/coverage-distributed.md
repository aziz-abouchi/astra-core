# Couverture distribuée dans Astra Core

La couverture distribuée mesure :

- les transitions de protocole,
- les messages,
- les migrations,
- les branches logiques.

## Types de couverture

### 1. Couverture réseau

- messages envoyés,
- messages reçus.

### 2. Couverture protocolaire

- transitions,
- états.

### 3. Couverture logique

- substitutions,
- branches.

### 4. Couverture cluster

- migrations,
- supervision.

## Visualisation

- heatmaps,
- graphes,
- timelines.
