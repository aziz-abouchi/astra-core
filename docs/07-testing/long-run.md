# Long-Run Testing dans Heaven Core

Les tests longue durée garantissent :

- stabilité,
- absence de fuite,
- robustesse.

## Objectifs

- exécutions prolongées,
- charge continue,
- simulation de production.

## Mécanismes

- rotation des fibres,
- rotation des capacités,
- rotation des nœuds,
- monitoring continu.

## Détection

- fuites mémoire,
- dérives logiques,
- instabilités réseau,
- saturation.

## Visualisation

- courbes longue durée,
- heatmaps,
- logs typés.
