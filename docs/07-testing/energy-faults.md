# Energy Fault Testing dans Heaven Core

Les fautes énergétiques permettent :

- de tester la frugalité,
- de détecter les pics,
- de valider les budgets.

## Types de fautes

- surcharge CPU,
- surcharge mémoire,
- surcharge réseau,
- oscillations énergétiques.

## Visualisation

- courbes énergie/temps,
- heatmaps,
- comparaisons.
