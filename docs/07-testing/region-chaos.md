# Region Chaos Testing dans Astra Core

Le region chaos testing permet :

- de tester la robustesse mémoire,
- de détecter les fuites,
- de valider les invariants.

## Types de chaos

- fragmentation,
- surcharge,
- libérations massives,
- migrations rapides.

## Visualisation

- heatmaps,
- timelines,
- métriques.
