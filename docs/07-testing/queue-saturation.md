# Tests de saturation de queues dans Astra Core

Ces tests mesurent :

- la saturation des queues,
- la résilience,
- la frugalité.

## Types de queues

- queues d’acteurs,
- queues réseau,
- queues logiques.

## Tests

### 1. Saturation progressive

- augmentation du débit,
- mesure de la latence.

### 2. Saturation extrême

- millions de messages,
- backpressure.

### 3. Saturation distribuée

- plusieurs nœuds,
- supervision.

## Visualisation

- heatmaps,
- courbes,
- timelines.
