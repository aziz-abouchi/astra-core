# Objectifs d’Heaven Core

- Offrir un langage concurrent et distribué sans data races.
- Intégrer un proof assistant minimal mais puissant.
- Garantir la frugalité : mémoire, CPU, énergie, temps.
- Fournir un runtime déterministe et mesurable.
- Supporter la transpilation vers Zig, C, WASM, JS.
- Proposer un modèle de capacités pour contrôler les effets.
- Permettre le type-driven development comme mode de travail naturel.
- Offrir une toolchain complète : REPL, profiler, debugger, transpiler.
