# Ownership linéaire dans Heaven Core

Heaven Core combine :

- ownership,
- linéarité,
- capacités,
- distribution.

## Principes

1. **Une ressource linéaire ne peut pas être copiée**
2. **Elle doit être consommée exactement une fois**
3. **Les transferts sont explicites**
4. **Les capacités reflètent l’ownership réel**
