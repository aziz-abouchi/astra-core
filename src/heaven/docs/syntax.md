# Syntaxe de HEAVEN

- Pattern matching directement dans la signature
- Guards avec `when`
- Récursion TCO implicite
- Accumulateurs implicites via paramètres par défaut
- Listes et tuples avec `[head | tail]` et `[]`
- Affections: `=` pour assignation initiale, `:=` pour réaffectation
- Comparaison: `==` ou `!=`
